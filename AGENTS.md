# AI エージェント基本方針

## 目的
リポジトリで作業するすべての AI エージェント共通の作業方針を定義します。

## 基本方針
- **会話と言語**: 日本語を使用する
- **コード内コメント**: 日本語で記載する
- **エラーメッセージ**: 英語で記載する
- **日本語と英数字の間**: 半角スペースを挿入する
- **コミット規約**: [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) に従い、説明は日本語で記載する

## 判断記録のルール
作業中の重要な判断については、以下の内容を明示する：
1. 判断内容の要約
2. 検討した代替案
3. 採用しなかった案とその理由
4. 前提条件・仮定・不確実性

## 開発手順（概要）
1. **プロジェクト理解**: 既存のコード構造、規約、技術スタックを分析する
2. **依存関係インストール**: 指定されたパッケージマネージャー（pnpm）を使用する
3. **変更実装**: プロジェクトのスタイルに合わせ、docstring を日本語で付与する
4. **検証**: `pnpm lint` や `pnpm compile` を実行し、品質を確認する

## セキュリティ / 機密情報
- API キー、トークン、パスワードなどの認証情報を Git にコミットしない
- ログに機密情報（個人情報、トークン等）を出力しない
- `.gitignore` で指定されたファイルを尊重する

## リポジトリ固有
- このプロジェクトは Discord Bot です。`discord.js` を使用しています。
- 設定は `data/config.json` で管理されます。
- `pnpm` をパッケージマネージャーとして使用します。
- `ncc` を使用してビルドされます。
